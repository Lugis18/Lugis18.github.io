La declaración <-!DOCTYPE HTML>; es una de algunas etiquetas que no se cierran, representa el tipo de documento y ayuda a los navegadores a mostrar las páginas web correctamente.

Todos los documentos HTML deben comenzar con una declaración de tipo de documento: .<-!DOCTYPE html><-/DOCTYPE html>. Indica el incio y final de un documento

La etiqueta <-head><-/head> de HTML sirve para contener metadatos, que es información sobre la página web, pero que no es visible

La etiqueta <-BODY><-/BODY>indica laparte visible del documento, lo que podrá ver usuarioel

Los enlaces HTML se definen con la etiqueta:<-a><-/a>

Las imágenes HTML se definen con la etiqueta.<-img>. El archivo de origen (), el texto alternativo (), y se proporcionan como atributos:src alt width height

Las etiquetas HTML <-ul>(Lista desordenada) y <-ol>(Lista ordenada) se usan para crear listas en una página web

Ejemplo de lista ordenada, aqui tendremos una lista enumerada

La etiqueta <-li> sirve para indicar los elementos internos de una lista HTML, ya sea no enumerada o lista enumerada
